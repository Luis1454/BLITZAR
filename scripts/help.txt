Targets:
  make run            Doctor check + build + deploy Qt + launch GUI
  make gui            Build + deploy Qt + launch GUI (no pre-check)
  make qt             Same as gui
  make all            Configure + build using current vars
  make test           Build then run ctest on BUILD_DIR
  make build-dev      Debug profile (BUILD_TESTS=ON)
  make build-prod     Release prod profile (deterministic path, dynamic modules off)
  make build-run      Release profile (BUILD_TESTS=OFF)
  make build-ci       Release profile (BUILD_TESTS=ON)
  make test-int       Configure/build/run standalone integration suite from tests/
  make doctor         Validate local toolchain/build artifacts via CMake script
  make deploy-qt      Manual Qt runtime/plugin deploy fallback for client host
  make run-server    Doctor check + run server mode via main binary
  make run-headless   Doctor check + run headless mode via main binary
  make clean          Remove build directory
  make clean-all      Remove all local build/output artifacts
  make re             Clean + full rebuild

Useful vars:
  BUILD_DIR=<dir>        Build directory for app targets (default: build)
  BUILD_TYPE=<type>      CMake build type (default: Release)
  BUILD_TESTS=ON|OFF     Toggle BLITZAR_BUILD_TESTS (default: ON)
  PROFILE=dev|prod       Build profile (default: dev)
  GENERATOR=<name>       CMake generator (default: Ninja)
  CUDA_ARCH=<arch>       CMAKE_CUDA_ARCHITECTURES (default: native)
  JOBS=<n>               Parallel build jobs (default: CMake default)
  INT_BUILD_DIR=<dir>    Build directory for standalone tests (default: build-integration)
  INT_BUILD_TYPE=<type>  Build type for standalone tests (default: Release)
  INT_TEST_REGEX=<regex> Optional ctest regex for test-int
  INT_TEST_REGEX_NO_SERVER=<regex> Default safe subset regex when SERVER_EXE is unavailable
  INT_TIMEOUT=<sec>      ctest timeout for test-int (default: 180)
  SERVER_EXE=<path>     Optional server executable path for real integration tests
                         If omitted and build/blitzar-server(.exe) is missing, integration_real tests are skipped
  CONFIG=<file>          Config file passed to client host (default: simulation.ini)
  GUI_MODULE=<name>      Client module name (default: qt)
  RUN_DOCTOR=0|1         Enable/disable doctor before run targets (default: 1)
  QT_DIR=<path>          Qt root path used by deploy tools
  VCPKG_TRIPLET=<triplet> vcpkg triplet for deps-graphics targets
  ARGS='...'             Extra args forwarded to runtime
